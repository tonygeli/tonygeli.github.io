composer 中版本号使用

~和^的意思很接近，在x.y的情况下是一样的都是代表x.y <= 版本号 < (x+1).0，但在版本号是x.y.z的情况下有区别，举个例子：

~1.2.3 代表 1.2.3 <= 版本号 < 1.3.0

^1.2.3 代表 1.2.3 <= 版本号 < 2.0.0
